##存储

  • 描述cookie ，session Storage ， localStorage 的区别

###cookie
1 用于客户端和服务器通信
2 有本地存储的功能就被借用
3 使用document.cookie (key,value)

##缺点

  • 存储太小,只有4kb
  • 所有http 请求都带着，会影响获取资源的效率
  • API简单，需要封装才能用document.cookie =
